# Strategic Forgetting Archive

## Purpose
This directory contains files, scripts, and documentation that are no longer in active use but may hold historical value or context.

## Retention Policy
-   **Trigger**: When a file is deprecated, it is moved here.
-   **Duration**: Files are retained for **30 days** after their last access or modification in this folder.
-   **Deletion**: If a file has not been "touched" (used/referenced) for 30 days, it is eligible for permanent deletion.
-   **Reset**: If you retrieve information from a file here, **MOVE IT BACK** to the active workspace or update its timestamp to restart the countdown.
